# @llm/mobile-app

The Mobile Web application for the LLM Chat interface.

## Features

- **Mobile First**: Touch-friendly interface optimized for smaller screens.
- **Responsive Layout**: Adaptive design that fits various mobile devices.
- **Performance**: Lightweight and fast loading for mobile networks.

## Development

```bash
pnpm dev:mobile
```

## Build

```bash
pnpm build:mobile
```

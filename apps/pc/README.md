# @llm/pc-app

The PC Web application for the LLM Chat interface.

## Features

- **Desktop Optimized**: Full-width layout with sidebar and panels.
- **Rich Interactions**: Drag-and-drop, keyboard shortcuts, and more.
- **Integrated Experience**: Combines `@llm/ui` and `@llm/store` for a complete chat solution.

## Development

```bash
pnpm dev:pc
```

## Build

```bash
pnpm build:pc
```

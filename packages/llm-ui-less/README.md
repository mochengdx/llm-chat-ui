# @llm/ui-less

A lightweight version of the LLM Chat UI library, using Less for styling.

## Features

- **Lightweight**: Optimized for performance with minimal dependencies.
- **Less Styling**: Uses Less for flexible and maintainable styling.
- **Core Components**: Includes essential chat components compatible with the main UI library.

## Installation

```bash
pnpm add @llm/ui-less
```

## Usage

```typescript
import { ChatMain } from "@llm/ui-less";
// ...
```

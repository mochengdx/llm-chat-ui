# @llm/ui (简体中文)

用于构建 LLM 聊天界面的综合 React UI 组件库。

## 功能特性

- **聊天组件**: 开箱即用的消息、输入区域和聊天列表组件。
- **Markdown 渲染**: 支持语法高亮和自定义指令的高级 Markdown 渲染。
- **主题定制**: 基于 Tailwind CSS 构建，易于定制并支持暗色模式。
- **响应式设计**: 完美适配桌面端和移动端设备。

## 安装

```bash
pnpm add @llm/ui
```

## 使用

```typescript
import { ChatMain } from '@llm/ui';

function App() {
  return <ChatMain />;
}
```

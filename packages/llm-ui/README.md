# @llm/ui

A comprehensive React UI component library for building LLM chat interfaces.

## Features

- **Chat Components**: Ready-to-use components for messages, input areas, and chat lists.
- **Markdown Rendering**: Advanced Markdown support with syntax highlighting and custom directives.
- **Theming**: Built with Tailwind CSS for easy customization and dark mode support.
- **Responsive**: Designed to work seamlessly on both desktop and mobile devices.

## Installation

```bash
pnpm add @llm/ui
```

## Usage

```typescript
import { ChatMain } from '@llm/ui';

function App() {
  return <ChatMain />;
}
```
